Le JavaScript est un langage de programmation script principalement utilisé dans des pages web interactives, c'est un langage orienté objet.

Le JavaScript à été inventé par Brendan Eich en 1995 sous le nom d'ECMAScript en juin 1997 dans le standard ECMA-262.

Le JavaScript est une implémentation d'ECMAScript, celle de Mozilla. l'implementation faite dans Internet Explorer de Microsoft se nomme "JScript".

Il est aussi employé dans les serverurs, avec par exemple Node.js ou Deno.

Du JavaScript fut notamment utilisé pour la page du qcm, pour mettre en place la selection des questions, et l'affichage de la réponse, plus un comptage du nombre de points.
