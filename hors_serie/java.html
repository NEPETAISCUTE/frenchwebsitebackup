Le Java est un langage de programmation orienté objet créé par James Gosling et Patrick Naughton, employés de Sun Microsystems, avec l'aide de Bill Joy, puis présenté officiellement au SunWorld en 1995.

J'ai classé le Java comme étant Hors Série car il se toruve que le fonctionnement de Java est assez unique, il se trouve que le Java est compilé dans un format de code machine intérmédiaire, avant d'en suite être interprété par une machine virtuelle (JVM) faisant donc abstraction du système d'exploitation.

Le Java reprends en partie le C++, mais le Java à évité certains des concepts les plus complexes, comme par exemple les pointeurs et références,
